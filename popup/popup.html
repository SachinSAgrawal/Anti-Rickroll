Anti Rickroll
Displays a warning when you navigate to a preset list of YouTube rickrolls (and Bitly links in the future) with an optional password to continue. Features also include the ability to report the current page as a rickroll, track statistics for blocked/bypassed links, edit and copy from the preset list of links, and disable the extension.
